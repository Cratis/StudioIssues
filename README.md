## 📋 About This Repository

This repository is the **public issue tracker** for [Cratis Studio](https://cratis.studio).

> ⚠️ **There is no code here.** All development happens in private repositories. This repo exists solely for community-driven issue tracking and feature requests.

Use this space to:

- 🐛 **Report bugs** you encounter while using Studio
- 💡 **Request features** or improvements you'd like to see
- 🗣️ **Discuss ideas** with the Cratis team and community

---

## 🚀 What is Cratis Studio?

[Cratis Studio](https://cratis.studio) is a visual development environment designed to make building event-sourced applications fast, intuitive, and enjoyable. It provides a rich UI on top of [Cratis Chronicle](https://github.com/Cratis/Chronicle) — letting you design, visualize, and manage your event-sourced systems without leaving your browser.

Visit **[cratis.studio](https://cratis.studio)** to learn more and get started.

## 🐛 Reporting Issues

When reporting a bug, please include as much of the following as possible:

- A clear and descriptive title
- Steps to reproduce the issue
- What you expected to happen
- What actually happened
- Your Studio version and browser/OS details
- Any relevant screenshots or error messages

👉 [Open a bug report](https://github.com/Cratis/StudioIssues/issues/new?labels=bug)

---

## 💡 Requesting Features

Have an idea that would make Studio better? We'd love to hear it!

When requesting a feature, please describe:

- The problem you're trying to solve
- Your proposed solution or idea
- Any alternatives you've considered

👉 [Open a feature request](https://github.com/Cratis/StudioIssues/issues/new?labels=enhancement)
